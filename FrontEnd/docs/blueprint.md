# **App Name**: Frontend Burger

## Core Features:

- Customer Registration: Allow new customers to create accounts via a registration form.
- Admin Registration: Allow new admins to create accounts via a registration form.
- Customer Login: Authenticate existing customers via a login form and JWT.
- Admin Login: Authenticate admins via a login form, verifying role through JWT.
- Order History Display: Display the order history for authenticated customers.
- Product Addition: Enable admins to add new products to the menu using a product form.
- Health Check Display: Simple page indicating if backend service is online

## Style Guidelines:

- Primary color: Deep red (#8B0000) to evoke a sense of richness and appetite appeal, referencing the ingredients.
- Background color: Soft beige (#F5F5DC), offering a clean and neutral backdrop that complements the vibrancy of the primary color and keeps the overall feel modern.
- Accent color: Mustard yellow (#FFDB58) to highlight key elements such as buttons and interactive components. It is analogous to red but has its own brighter, livelier feel, perfect to highlight CTAs.
- Headline font: 'Belleza', a humanist sans-serif with personality, for headlines and short chunks of text. Body font: 'Alegreya' a humanist serif with an elegant, intellectual, contemporary feel, perfect for readability.
- Use simple, outline-style icons for navigation and product categories.
- Use a clean, grid-based layout with clear separation between sections. Favor full-width sections, in a one-page format with sticky headers
- Use subtle transitions and animations on hover and click events to provide feedback to user interactions.
